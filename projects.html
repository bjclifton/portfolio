TCP Protocol Implementation

This project involved building a robust, simulated version of the Transmission Control Protocol (TCP) from the ground up to provide deep insight into reliable transport layer operations.

Key Technical Aspects

  • Connection Management: Implemented the three-way handshake for connection establishment and termination.
  • Reliability: Developed selective repeat mechanisms for efficient packet retransmission and error correction.
  • Flow Control: Integrated a sliding window implementation to prevent buffer overflow and manage data flow between sender and receiver.
  • Congestion Control: Successfully implemented a simplified congestion control algorithm (similar to Tahoe/Reno) that dynamically adjusts the send rate based on network feedback.

Impact

This project solidified my understanding of fundamental network reliability engineering principles, focusing on the core challenges of sequencing, acknowledgment, and resource management.

Skills: C++, Networking, Socket Programming, Reliable Transport

Network Performance Tool (iPerf Clone)

This command-line tool was developed to measure network throughput and latency, acting as both a client and a server to diagnose bandwidth limitations and latency issues.

Functionality and Implementation

  • Protocol Support: Measures throughput using both TCP (for sustained transfer rates) and UDP (for analyzing packet loss and jitter).
  • Concurrency: Supports multithreading to run concurrent connections, simulating real-world high-load scenarios.
  • Low-Level Networking: Leverages Linux sockets and C++ for efficient, low-level network communication.

Result

The tool accurately calculates and reports essential metrics like bits per second (Mbps) and round-trip time (RTT), offering a fast and reliable way to benchmark network performance.

Skills: C++, Linux, Multithreading, Socket Programming

Certificate Automation Tool

The Challenge: Manual renewal and distribution of SSL/TLS certificates across over 30 enterprise services was inefficient, error-prone, and risked service downtime due to expiry.

The Solution

  • Automated Lifecycle: Designed a robust .NET/C# backend service to fully automate the certificate lifecycle.
  • Integration: Uses REST APIs to interface with external certificate authorities and internal application load balancers.
  • Monitoring: Continuously monitors expiration dates and orchestrates the renewal, validation, and deployment process automatically.

Impact

By automating the process, the tool reduced the manual overhead for platform engineers by approximately 80% and eliminated the risk of downtime, significantly improving the organization's security posture.

Skills: .NET/C#, REST APIs, Automation, Enterprise Security
